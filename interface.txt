============================================================================
SignIn(로그인)

*는 필수

- request 
{
    *email: String,
    *password: String,
}

- response

성공
Http Status - 200 (OK)
{
    code: "SU",
    message: "SUCCESS.",
    token: "jwt...",
    expiredDate: 123456789
}

실패

- 필수 정보 미입력

- 로그인 실패
Http Status - 401 (Unauthorized)
{
    code: "SF",
    message:"SignIn Failed."
}

- 데이터베이스 에러
Http Status - 500 (Internal Server Error)
{
    code: "DE",
    message:"Database Error."
}

signUp (회원가입)

- request
{
    *email: String,
    *password: String,
    *nickname: String,
    *telNumber: String,
    *address: String,
    addressDetail: String,
}

- response

Http Status - 200 (OK)
성공
{
    code: "SU",
    message: "SUCCESS."
}

실패

- 필수 정보 미입력 / 이메일 포멧 불일치 / 비밀번호 8자 미만 / 전화번호 포멧 불일치 

- 이메일 중복
Http Status - 400 (Bad Request)
{
    code: "EE",
    message:"Existed Email."
}

- 데이터베이스 에러
Http Status - 500 (Internal Server Error)
{
    code: "DE",
    message:"Database Error."
}
